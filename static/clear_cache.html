🔄 브라우저 캐시 문제 해결

문제 상황

브라우저가 이전 버전의 JavaScript를 캐싱하고 있어 수정사항이 반영되지 않습니다.

콘솔에서 chunk: undefined가 나타나는 것이 증거입니다.

해결 방법

아래 버튼들을 클릭하여 새 버전의 페이지로 이동하세요:

📺 LG QNED TV (새 버전) 💧 LG 정수기 (새 버전) 👕 LG 건조기 (새 버전) ❄️ LG 에어컨 (새 버전)

추가 해결 방법

  1. 강력 새로고침: Ctrl + Shift + R (Windows) / Cmd + Shift + R (Mac)
  2. 개발자 도구: F12 → Network 탭 → Disable cache 체크
  3. 시크릿 모드: Ctrl + Shift + N (Chrome) / Ctrl + Shift + P (Firefox)
  4. 캐시 삭제: Ctrl + Shift + Delete → 캐시된 이미지 및 파일 선택 → 삭제
